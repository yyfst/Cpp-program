* ### 思考题

1.术语prompt的含义是什么？  
&emsp;答：提示符的意思，在代码中输出一段提示信息，称为提示符.

2.哪种操作符用于读取值并存入变量中？  
&emsp;答:`>>`，如`cin>>a`，表示将输入的信息存入a中.
